Estructura Básica de Tablas HTML

¿Qué es una tabla HTML?

Una tabla HTML es una estructura que permite organizar datos en filas y columnas.

Elementos básicos de una tabla

Elemento Descripción
<table> Contenedor principal que define la tabla
<tr> Define una fila dentro de la tabla
<th> Define una celda de encabezado (header cell)
<td> Define una celda de datos (table data)

Ejemplo de tabla básica

Fila 1, Columna 1 Fila 1, Columna 2
Fila 2, Columna 1 Fila 2, Columna 2

Código HTML del ejemplo anterior

<table border="1">
    <tr>
        <td>Fila 1, Columna 1</td>
        <td>Fila 1, Columna 2</td>
    </tr>
    <tr>
        <td>Fila 2, Columna 1</td>
        <td>Fila 2, Columna 2</td>
    </tr>
</table>

Elementos adicionales para tablas

Elemento Descripción
<caption> Proporciona un título para la tabla
<thead> Agrupa el contenido del encabezado
<tbody> Agrupa el contenido del cuerpo
<tfoot> Agrupa el contenido del pie

Atributos importantes para tablas

  • border: Define el grosor del borde (en píxeles)
  • cellpadding: Espacio entre el contenido y el borde de la celda
  • cellspacing: Espacio entre las celdas
  • colspan: Número de columnas que ocupa una celda
  • rowspan: Número de filas que ocupa una celda

Nota: Aunque estos atributos siguen funcionando, en HTML5 se recomienda utilizar CSS para controlar la apariencia de las tablas.
