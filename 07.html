Ejemplos de Hipervínculos en HTML

Los hipervínculos (o enlaces) son elementos fundamentales en HTML que permiten navegar entre páginas web.

1. Hipervínculo básico

El elemento <a> (anchor) crea un hipervínculo. El atributo href especifica la URL de destino.

Ejemplo en acción: Visitar Ejemplo.com

<a href="https://www.ejemplo.com">Visitar Ejemplo.com</a>

2. Hipervínculo que abre en nueva pestaña

Usando el atributo target="_blank", el enlace se abrirá en una nueva pestaña o ventana.

Ejemplo en acción: Abrir Google en nueva pestaña

<a href="https://www.google.com" target="_blank">Abrir Google en nueva pestaña</a>

3. Hipervínculo a una sección de la misma página

Podemos enlazar a secciones específicas de la misma página usando el atributo id.

Ejemplo en acción: Ir a la sección de destino

<a href="#seccion-destino">Ir a la sección de destino</a>

4. Hipervínculo a un archivo local

Podemos enlazar a archivos dentro de nuestro propio sitio web usando rutas relativas.

Ejemplo en acción: Abrir documento PDF (enlace simulado)

<a href="documento.pdf">Abrir documento PDF</a>

5. Hipervínculo de correo electrónico

Usando mailto: en el atributo href, podemos crear enlaces que abren el cliente de correo.

Ejemplo en acción: Enviar correo a info@ejemplo.com

<a href="mailto:info@ejemplo.com">Enviar correo a info@ejemplo.com</a>

6. Hipervínculo con título (tooltip)

El atributo title muestra un texto emergente cuando el usuario pasa el cursor sobre el enlace.

Ejemplo en acción: Visitar Wikipedia

<a href="https://www.wikipedia.org" title="Enciclopedia libre en línea">Visitar Wikipedia</a>
