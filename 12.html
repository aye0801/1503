Etiquetas Avanzadas en HTML con Multimedia

Explora ejemplos funcionales de etiquetas avanzadas de HTML con contenido multimedia e interactivo. Aprende sobre su uso, características y alternativas modernas.

1. <iframe> - Incrustar contenido externo

Permite incrustar otra página web dentro de la página actual. Es útil para mostrar contenido de terceros como mapas, videos o documentos.

Características principales:

  • Permite la integración de contenido externo
  • Puede tener dimensiones personalizadas
  • Soporta atributos de seguridad como sandbox

Ejemplo funcional:

Código de ejemplo:

<iframe src="https://ejemplo.com" width="400" height="300"></iframe>

2. <script> - Integración de JavaScript

Permite agregar código JavaScript directamente en el documento HTML o enlazar archivos externos.

Características principales:

  • Ejecuta código JavaScript en el navegador
  • Puede estar en línea o referenciar archivos externos
  • Soporta atributos como async y defer para controlar la carga

Ejemplo funcional:

Código de ejemplo:

<script>
function miFuncion() {
  alert('¡Hola mundo!');
}
</script>

3. <style> - Estilos CSS internos

Permite definir estilos CSS directamente dentro del documento HTML.

Características principales:

  • Define estilos específicos para la página
  • Útil para estilos que no se reutilizan en otras páginas
  • Se puede usar junto con hojas de estilo externas

Ejemplo funcional:

Código de ejemplo:

<style>
.mi-clase {
  color: red;
  font-size: 16px;
}
</style>

4. <embed> - Contenido multimedia incrustado

Incrusta contenido externo como PDFs, SWF u otros tipos de archivos. Aunque es una etiqueta antigua, aún se utiliza en algunos contextos.

Características principales:

  • Incrusta contenido interactivo o multimedia
  • Requiere que el navegador tenga el plugin adecuado
  • En muchos casos, se prefiere usar iframe u object

Ejemplo funcional (PDF usando iframe como alternativa):

Código de ejemplo:

<embed src="documento.pdf" width="600" height="400" type="application/pdf">

5. <audio> - Reproductor de audio

Etiqueta moderna para incrustar y reproducir archivos de audio en una página web.

Características principales:

  • Reproduce archivos de audio sin necesidad de plugins
  • Soporta múltiples formatos (MP3, WAV, OGG)
  • Controles personalizables para reproducción

Ejemplo funcional (reemplazo de bgsound obsoleto):

Nota: El atributo autoplay se ha eliminado para mejor experiencia de usuario.

Código de ejemplo:

<audio src="audio.mp3" controls>
  Tu navegador no soporta el elemento audio.
</audio>

6. <object> - Objetos incrustados (reemplazo de applet)

Etiqueta moderna para incrustar objetos externos como aplicaciones Java, PDFs o contenido multimedia.

Características principales:

  • Reemplaza a la etiqueta <applet> obsoleta
  • Puede incrustar diversos tipos de contenido
  • Más seguro y estándar que las alternativas antiguas

Ejemplo funcional (simulación):

Ejemplo moderno usando <object> para incrustar un PDF:

Código de ejemplo:

<object data="applet.jar" type="application/x-java-applet">
  <p>Tu navegador no soporta Java applets.</p>
</object>

7. <iframe> como reemplazo de <frame>

Los framesets y frames han sido reemplazados por iframes en el HTML moderno.

Características principales:

  • Los framesets dividían la ventana en varias secciones
  • Los iframes son más flexibles y modernos
  • Mejor compatibilidad con dispositivos móviles

Ejemplo funcional (simulación de frame con iframe):

Código de ejemplo (frameset obsoleto):

<!-- Código obsoleto -->
<frameset cols="25%,75%">
  <frame src="menu.html">
  <frame src="contenido.html">
</frameset>

8. Alternativas modernas a <marquee>

La etiqueta <marquee> está obsoleta y se recomienda usar CSS o JavaScript para animaciones.

Características principales:

  • <marquee> creaba texto en movimiento
  • Ahora se usa CSS animations o JavaScript
  • Las alternativas modernas son más eficientes y accesibles

Ejemplo funcional (marquee obsoleto vs. alternativa CSS):

Ejemplo con <marquee> (obsoleto pero aún funciona en algunos navegadores):

Alternativa moderna con CSS (requeriría estilos):

Código de ejemplo (alternativa CSS):

<style>

  100% { transform: translateX(-100%); }
}
</style>

9. <img> - Imágenes en la web

Etiqueta fundamental para mostrar imágenes en páginas web.

Características principales:

  • Soporta múltiples formatos (JPEG, PNG, GIF, SVG, WebP)
  • Atributos importantes: src, alt, width, height
  • Nuevos atributos como loading="lazy" para carga diferida

Ejemplo funcional:

Imágenes de ejemplo cargadas desde fuentes externas:

Código de ejemplo:

<img src="imagen.jpg" alt="Descripción de la imagen" width="400" height="300">

10. <video> - Reproductor de video

Etiqueta moderna para incrustar y reproducir videos en una página web.

Características principales:

  • Reproduce video sin necesidad de plugins como Flash
  • Soporta múltiples formatos (MP4, WebM, OGG)
  • Controles de reproducción, volumen y pantalla completa

Ejemplo funcional:

Código de ejemplo:

<video width="640" height="360" controls>
  <source src="video.mp4" type="video/mp4">
  <source src="video.webm" type="video/webm">
  Tu navegador no soporta el elemento de video.
</video>

11. <canvas> - Gráficos y animaciones

Elemento para dibujar gráficos, animaciones y visualizaciones mediante JavaScript.

Características principales:

  • Superficie de dibujo mediante API de JavaScript
  • Ideal para gráficos, juegos y visualizaciones complejas
  • Requiere scripting para mostrar contenido

Ejemplo funcional:

Tu navegador no soporta el elemento canvas.

Código de ejemplo:

<canvas id="lienzo" width="400" height="200"></canvas>
<script>
  var canvas = document.getElementById('lienzo');
  var ctx = canvas.getContext('2d');
  ctx.fillRect(0, 0, 400, 200);
</script>

12. <svg> - Gráficos vectoriales

Elemento para crear gráficos vectoriales escalables directamente en HTML.

Características principales:

  • Gráficos que no pierden calidad al escalarse
  • Se puede manipular con CSS y JavaScript
  • Ideal para iconos, gráficos simples y animaciones

Ejemplo funcional:

Código de ejemplo:

<svg width="100" height="100">
  <circle cx="50" cy="50" r="40" fill="red" />
</svg>
