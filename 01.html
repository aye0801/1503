HISTORIA DE HTML

1991: Tim Berners-Lee crea la primera versión de HTML con solo 18 etiquetas para compartir documentos científicos en la web. 1995: Se lanza HTML 2.0, el primer estándar oficial. 1997-1999: Aparecen HTML 3.2 y 4.01, con soporte para estilos (CSS), scripts (JavaScript) y tablas. 2000s: Se intenta reemplazar HTML con XHTML, pero fracasa por su rigidez. 2014: Llega HTML5, una gran actualización con soporte para video, audio, APIs y etiquetas semánticas. 2019 en adelante: HTML se convierte en un "estándar vivo" (Living Standard), actualizado continuamente por el grupo WHATWG.
